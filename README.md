# TechFlow Solutions - Sistema de Gerenciamento de Tarefas

## 📌 Objetivo do Projeto
Este projeto tem como objetivo simular um sistema de gerenciamento de tarefas utilizando metodologias ágeis, permitindo acompanhar atividades, priorizar demandas e melhorar o fluxo de trabalho de uma equipe.

## 🏢 Contexto
A TechFlow Solutions desenvolveu este sistema para uma startup de logística que precisava organizar suas tarefas em tempo real e melhorar a produtividade da equipe.

## ⚙️ Metodologia Ágil
Foi utilizada a metodologia Kanban, com organização das tarefas em:
- To Do
- In Progress
- Done

## 💻 Funcionalidades
- Sistema básico de gerenciamento de tarefas (CRUD)
- Controle de status das tarefas
- Estrutura organizada em repositório GitHub

## 🔁 Mudança de Escopo
Durante o desenvolvimento, foi adicionada uma nova funcionalidade: priorização de tarefas críticas, permitindo melhor controle das atividades mais urgentes.

## 🧪 Testes Automatizados
Foi utilizado GitHub Actions para executar testes automatizados, garantindo a qualidade do código e a validação das funcionalidades principais.

## 📊 Ferramentas Utilizadas
- GitHub
- GitHub Projects (Kanban)
- GitHub Actions
- Python (ou linguagem do seu projeto)

## 📌 Conclusão
O projeto demonstra a aplicação prática de engenharia de software, integração contínua e metodologias ágeis no desenvolvimento de sistemas reais.
